# Керування

| -------------- | ----------------- | --------------- |

| Присідання     | C                 | Кнопка Л. Стіка |
| Деш            | Shift             | Кнопка П. Стіка |
| Взаємодія      | E                 | Y               |
| Атака          | Ліва кнопка миші  | RT              |
| Додаткова дія  | Права кнопка миші | LT              |
| Гак кішка      | R                 | LB              |
| Режим демона   | F                 | RB              |
| Спец уміння    | Q                 | X               |
| Вибір с.уміння | 1, 2, 3, 4        | Хрестовина      |
| Провокація     | Z                 | B               |
| Пауза          | Esc               | Start           |
| Дод. меню      | U                 | Select          |
